# Wedding site — Аскар & Анель

Готовый одностраничный wedding invitation на чистом HTML/CSS/JS с адаптивной вёрсткой, галереей, календарём и RSVP-формой, которая отправляет данные в Telegram через безопасный серверless-прокси.

## Что внутри

- `index.html` — разметка сайта
- `styles.css` — mobile-first стили
- `script.js` — клиентская валидация и AJAX-отправка формы
- `worker.js` — Cloudflare Worker для безопасной отправки в Telegram Bot API
- `assets/*.webp` — лёгкие декоративные WebP-заглушки под hero и галерею

## Что заменить перед публикацией

В `index.html`:
- название ресторана и адрес
- ссылку в блоке карты (`href="https://go.2gis.com/"`) на реальную route-ссылку 2GIS
- при необходимости текст приглашения

В `assets/`:
- замените `hero.webp`, `gallery-1.webp`, `gallery-2.webp` на ваши фотографии
- оптимальные форматы: WebP, разумное сжатие, ширина hero 1600–2000px, галерея 1000–1600px

В `script.js`:
- при необходимости поменяйте `FORM_ENDPOINT`, если Worker размещён не на `/api/rsvp`

## Подключение Cloudflare Worker

### 1) Создайте Worker
Разместите `worker.js` как отдельный Worker или через Pages Functions/Workers route.

### 2) Добавьте секреты и переменные
Добавьте:
- `BOT_TOKEN` — токен вашего Telegram-бота (**secret**)
- `CHAT_ID` — chat_id, куда бот будет отправлять ответы
- `ALLOWED_ORIGIN` — домен сайта, например `https://wedding.example.com`

### 3) Настройте маршрут
Привяжите Worker к пути:
- `https://your-domain.com/api/rsvp`

### 4) Проверьте запрос
Форма на фронтенде отправляет JSON:
```json
{
  "name": "Алия",
  "guests": "Ерлан, Аружан",
  "attendance": "Қуана келемін",
  "comment": "Күтеміз!",
  "phone": "+77001234567",
  "sentAt": "2026-03-24T12:00:00.000Z",
  "page": "https://your-domain.com/"
}
```

## Локальный запуск

Откройте `index.html` в браузере для проверки интерфейса.
Для теста формы нужен рабочий `/api/rsvp` endpoint.

## Проверка перед релизом

- нет горизонтального скролла на 320px
- hero корректно обрезается по `cover`
- кнопки и поля удобно нажимаются пальцем
- ссылка карты открывается в новой вкладке
- форма показывает ошибки до отправки
- Worker отвечает JSON и сообщение приходит в Telegram
- реальные изображения заменили декоративные заглушки
