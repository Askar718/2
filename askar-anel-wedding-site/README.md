# Wedding site v3 — Аскар & Анель

Обновлённая mobile-first версия свадебного сайта в более ярком, тёплом и цветочном стиле.

## Что изменено
- акцент на смартфоны: крупные кнопки, вертикальный ритм, липкая нижняя CTA-кнопка
- обновлённая тёплая цветочная палитра
- более мягкие reveal-анимации на скролле
- добавлен countdown
- 2GIS-ссылка уже подставлена: `https://go.2gis.com/9vStB`
- декоративные WebP-фоны перерисованы в более воздушном floral-настроении

## Файлы
- `index.html` — разметка
- `styles.css` — mobile-first стили
- `script.js` — reveal, countdown, валидация и AJAX-отправка
- `worker.js` — безопасный прокси для Telegram Bot API
- `assets/*.webp` — лёгкие декоративные изображения

## Что ещё можно заменить
- реальные фотографии пары вместо декоративных floral-изображений
- точное название ресторана и адрес, если хотите вывести их текстом
- домен Worker в `FORM_ENDPOINT`, если он не на `/api/rsvp`
